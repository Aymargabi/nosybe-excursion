# Nosy Be Excursions – Booking Website

## Description
Nosy Be Excursions est un site web touristique permettant aux visiteurs de découvrir et réserver des excursions à Nosy Be (Hell-Ville), Madagascar.

Le site présente différentes activités touristiques populaires de l'île et offre une interface simple pour consulter les excursions et planifier une visite.

## Fonctionnalités
- Page d’accueil avec présentation de Nosy Be
- Liste des excursions disponibles
- Carte interactive pour localiser les activités touristiques
- Navigation dynamique entre les pages
- Interface moderne et responsive

## Technologies utilisées
- HTML5
- CSS3
- JavaScript
- Leaflet.js (carte interactive)
- Google Fonts
